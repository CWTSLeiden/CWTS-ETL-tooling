Databasedocumentatiegenerator.exe

Purpose:

This program generates a PDF file with documentaiotn about a database.

Execute:

To execute this program it must be called from a batch file or command line with the correct parameters

Parameters:

database (mandatory)
The name of the database to generate the documentation for.

server (mandatory)
The name of the database server

outputfolder (mandatory)
The full path to the folder where the PDF file is stored

imagefolder (mandatory)
The full path to the folder where the images used for the documentation are stored.

logfolder (mandatory)
The full path to the folder where the error file is saved in case of an error. If no error occurs this
folder remains empty.
